
# Study Tracker (Java)

A simple daily PCMB study tracking console application.

## Features
- Planned vs actual study tracking
- Subject-wise tracking (Physics, Chemistry, Maths, Biology)
- Strongest and weakest subject detection
- Completion percentage capped at 100%
- Clean console output with feedback

## Built With
Java (Developed using BlueJ)
